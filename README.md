## Installation

Add this line to your application's Gemfile:

    gem 'jscal2-rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jscal2-rails

## Usage
```
/*
 *= require jscal2/jscal2
*/
``

```
